== Description ==

Elara is a beautiful, featured packed blogging theme.

* Front Page Banner
* Front Page Posts Slider
* Front Page Featured Posts
* Front Page Highlight Post
* Image and Text Logos
* Background Image
* Background Color
* Optional Full Width Posts and Pages
* Special Category Template
* Two-In-A-Row Blog Feed Layout
* Mixed Format (Two Small Third Full) Blog Feed Layout
* Integration with Recent Posts Widget With Thumbnails, MailPoet
* Easy to Create Social Media Icon Menus
* Header and Footer Area Widgets
